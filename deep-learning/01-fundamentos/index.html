    1.2. La Neurona Artificial (Perceptr贸n)

    La unidad b谩sica de una red neuronal es la neurona artificial, inspirada (de forma simplificada) en las neuronas biol贸gicas.

    Funcionamiento

    1. Entradas: Recibe valores \(x_1, x_2, ..., x_n\)
    2. Pesos: Cada entrada tiene un peso asociado \(w_1, w_2, ..., w_n\)
    3. Suma ponderada: \(z = \sum_{i=1}^{n} w_i \cdot x_i + b\) (donde \(b\) es el bias)
    4. Activaci贸n: \(a = f(z)\) (funci贸n de activaci贸n)
    5. Salida: El valor \(a\) se pasa a la siguiente capa

    Representaci贸n Matem谩tica

    \[a = f\left(\sum_{i=1}^{n} w_i x_i + b\right) = f(W^T X + b)\]
    import numpy as np

def neurona(X, W, b, activacion='sigmoid'):
    """
    Simula una neurona artificial.

    X: vector de entradas
    W: vector de pesos
    b: bias
    """
    # Suma ponderada
    z = np.dot(W, X) + b

    # Funci贸n de activaci贸n
    if activacion == 'sigmoid':
        a = 1 / (1 + np.exp(-z))
    elif activacion == 'relu':
        a = np.maximum(0, z)
    elif activacion == 'tanh':
        a = np.tanh(z)

    return a

    Funci贸n F贸rmula Rango Uso

    驴Por qu茅 ReLU es Popular?

    • Evita el problema del gradiente desvaneciente: La derivada es 1 para valores positivos.
    • Computacionalmente eficiente: Solo comparaci贸n y selecci贸n.
    • Promueve dispersi贸n: Muchas neuronas producen 0, creando representaciones dispersas.

    1.5. Forward Propagation

    Es el proceso de pasar los datos a trav茅s de la red para obtener una predicci贸n.

    import numpy as np

def forward_propagation(X, pesos, biases, activaciones):
    """
    Realiza forward propagation a trav茅s de la red.
    """
    activacion = X
    activaciones_cache = [X]

    for i, (W, b, func_act) in enumerate(zip(pesos, biases, activaciones)):
        z = np.dot(activacion, W) + b

        if func_act == 'relu':
            activacion = np.maximum(0, z)
        elif func_act == 'sigmoid':
            activacion = 1 / (1 + np.exp(-z))
        elif func_act == 'softmax':
            exp_z = np.exp(z - np.max(z))
            activacion = exp_z / exp_z.sum()

        activaciones_cache.append(activacion)

    return activacion, activaciones_cache

    1.6. Funci贸n de P茅rdida (Loss Function)

    La funci贸n de p茅rdida mide qu茅 tan lejos est谩n las predicciones de los valores reales.

    Funciones Comunes

    Mean Squared Error (MSE) - Para regresi贸n:

    \[MSE = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2\]

    Binary Cross-Entropy - Para clasificaci贸n binaria:

    \[BCE = -\frac{1}{n}\sum_{i=1}^{n}[y_i \log(\hat{y}_i) + (1-y_i)\log(1-\hat{y}_i)]\]

    Categorical Cross-Entropy - Para clasificaci贸n multiclase:

    \[CCE = -\sum_{i=1}^{n}\sum_{c=1}^{C}y_{i,c}\log(\hat{y}_{i,c})\]
    import numpy as np

def mse(y_true, y_pred):
    return np.mean((y_true - y_pred) ** 2)

def binary_crossentropy(y_true, y_pred):
    epsilon = 1e-15  # Evitar log(0)
    y_pred = np.clip(y_pred, epsilon, 1 - epsilon)
    return -np.mean(y_true * np.log(y_pred) + (1 - y_true) * np.log(1 - y_pred))

    1.7. Backpropagation

    El Backpropagation es el algoritmo que calcula los gradientes de la funci贸n de p茅rdida con respecto a cada peso, usando la regla de la cadena.

    Proceso

    1. Forward pass: Calcular predicciones.
    2. Calcular p茅rdida: Comparar predicci贸n con valor real.
    3. Backward pass: Calcular gradientes desde la salida hacia la entrada.
    4. Actualizar pesos: Usar los gradientes para ajustar los pesos.

    Regla de la Cadena

    \[\frac{\partial L}{\partial w} = \frac{\partial L}{\partial a} \cdot \frac{\partial a}{\partial z} \cdot \frac{\partial z}{\partial w}\]

    Donde: - \(L\) es la p茅rdida - \(a\) es la activaci贸n - \(z\) es la suma ponderada - \(w\) es el peso

    1.8. Optimizadores

    Los optimizadores actualizan los pesos bas谩ndose en los gradientes calculados.

    Gradient Descent

    \[w_{nuevo} = w_{actual} - \eta \cdot \nabla L\]

    Donde \(\eta\) es el learning rate (tasa de aprendizaje).

    Variantes Populares

    Optimizador Descripci贸n Cu谩ndo usarlo
    SGD Actualiza con mini-batches Baseline, simple
    SGD + Momentum A帽ade "inercia" a las actualizaciones Acelera convergencia
    RMSprop Adapta learning rate por par谩metro Datos no estacionarios
    Adam Combina Momentum + RMSprop Default recomendado
    AdamW Adam con weight decay correcto Estado del arte

    1.9. Ejemplo Completo: Clasificaci贸n con Keras

from tensorflow.keras.layers import Dense, Input, Dropout

    Dense(64, activation='relu'),
    Dropout(0.3),
    Dense(32, activation='relu'),
    Dropout(0.3),

    epochs=50,
    batch_size=32,

# Visualizar entrenamiento

plt.subplot(1, 2, 1)

plt.title('Loss')
plt.legend()

plt.subplot(1, 2, 2)
plt.plot(historia.history['accuracy'], label='Train')
plt.plot(historia.history['val_accuracy'], label='Validation')
plt.title('Accuracy')
plt.legend()

plt.tight_layout()
plt.show()
