    游닄 Unidad 5. Estructuras de Datos

    Python ofrece varias estructuras de datos incorporadas para almacenar colecciones de elementos. Las principales son: listas, tuplas, diccionarios y conjuntos (sets).

    5.1. Listas

    Las listas son colecciones ordenadas y modificables de elementos. Pueden contener elementos de diferentes tipos.

    Creaci칩n de Listas

    # Lista vac칤a

    Acceso a Elementos

    frutas = ["manzana", "naranja", "pl치tano", "uva", "pera"]

print(frutas[1])   # naranja
print(frutas[4])   # pera (칰ltimo)

# 칈ndices negativos (desde el final)
print(frutas[-1])  # pera (칰ltimo)
print(frutas[-2])  # uva (pen칰ltimo)

# Slicing (cortar)

print(frutas[:3])     # ['manzana', 'naranja', 'pl치tano']

print(frutas[::2])    # ['manzana', 'pl치tano', 'pera'] (cada 2)

    Modificar Elementos

# Cambiar varios elementos con slicing

    M칠todos de Listas

    A침adir Elementos

    B칰squeda y Conteo

    Ordenaci칩n

    5.2. Tuplas

    Las tuplas son colecciones ordenadas e inmutables (no se pueden modificar).

    Creaci칩n de Tuplas

    # Tupla vac칤a
tupla_vacia = ()
otra_vacia = tuple()

# Tupla con elementos
coordenadas = (10, 20)
datos = ("Ana", 25, "Madrid")
mixta = (1, "hola", 3.14)

# Tupla de un solo elemento (necesita coma)
un_elemento = (42,)  # Con coma
no_tupla = (42)      # Esto es solo un int

# Sin par칠ntesis (empaquetado)
punto = 5, 10
print(punto)  # (5, 10)

    Acceso a Elementos

    colores = ("rojo", "verde", "azul", "amarillo")

# Igual que las listas
print(colores[0])     # rojo
print(colores[-1])    # amarillo
print(colores[1:3])   # ('verde', 'azul')

# Desempaquetado
x, y, z = (1, 2, 3)
print(x, y, z)  # 1 2 3

# Desempaquetado con *
primero, *resto = (1, 2, 3, 4, 5)
print(primero)  # 1
print(resto)    # [2, 3, 4, 5]

*inicio, ultimo = (1, 2, 3, 4, 5)
print(inicio)  # [1, 2, 3, 4]
print(ultimo)  # 5

    Inmutabilidad

    tupla = (1, 2, 3)

# NO se puede modificar
# tupla[0] = 100  # Error: TypeError

# Pero se puede crear una nueva
nueva = tupla + (4, 5)
print(nueva)  # (1, 2, 3, 4, 5)

    M칠todos de Tuplas

    Las tuplas tienen menos m칠todos que las listas (por ser inmutables):

    numeros = (1, 2, 3, 2, 4, 2, 5)

# count() - Contar ocurrencias

# index() - Encontrar 칤ndice
print(numeros.index(4))  # 4

    Cu치ndo Usar Tuplas

    • Datos que no deben cambiar (coordenadas, configuraci칩n).
    • Claves de diccionarios (las listas no pueden ser claves).
    • Retornar m칰ltiples valores de una funci칩n.
    • Son m치s eficientes que las listas.
    # Retornar m칰ltiples valores

    cociente = a // b
    resto = a % b
    return cociente, resto  # Devuelve tupla

# Desempaquetando directamente

print(f"Cociente: {c}, Resto: {r}")

    5.3. Diccionarios

    Los diccionarios almacenan pares clave-valor. Son muy 칰tiles para representar datos estructurados.

    Creaci칩n de Diccionarios

    "nombre": "Ana",

    M칠todos de Diccionarios

    persona = {"nombre": "Ana", "edad": 30, "ciudad": "Madrid"}

    5.4. Conjuntos (Sets)

    Los sets son colecciones desordenadas de elementos 칰nicos.

    Creaci칩n de Sets

    # Set vac칤o (no se puede usar {}, eso crea un diccionario)

    Operaciones B치sicas

    frutas = {"manzana", "naranja", "pl치tano"}

# A침adir elemento

print(frutas)

# A침adir varios elementos
frutas.update(["pera", "kiwi"])
print(frutas)

# Eliminar elemento

frutas.discard("mango")   # No error si no existe

# pop() - Eliminar elemento aleatorio

# Verificar pertenencia

    Operaciones de Conjuntos

    A = {1, 2, 3, 4, 5}
B = {4, 5, 6, 7, 8}

# Uni칩n (elementos en A o B)
union = A | B  # o A.union(B)
print(union)   # {1, 2, 3, 4, 5, 6, 7, 8}

# Intersecci칩n (elementos en A y B)
interseccion = A & B  # o A.intersection(B)
print(interseccion)   # {4, 5}

# Diferencia (elementos en A pero no en B)
diferencia = A - B  # o A.difference(B)
print(diferencia)   # {1, 2, 3}

# Diferencia sim칠trica (elementos en A o B pero no en ambos)
sim_diferencia = A ^ B  # o A.symmetric_difference(B)
print(sim_diferencia)   # {1, 2, 3, 6, 7, 8}

    Verificar Subconjuntos

    A = {1, 2, 3}
B = {1, 2, 3, 4, 5}
C = {1, 2, 3}

# Subconjunto
print(A <= B)  # True (A es subconjunto de B)
print(A.issubset(B))  # True

# Subconjunto propio (m치s peque침o)
print(A < B)   # True
print(A < C)   # False (son iguales)

# Superconjunto
print(B >= A)  # True
print(B.issuperset(A))  # True

# Conjuntos disjuntos (sin elementos comunes)
X = {1, 2, 3}
Y = {4, 5, 6}
print(X.isdisjoint(Y))  # True

    Ejemplo: Eliminar Duplicados

    # Eliminar duplicados de una lista

print(lista_ordenada)  # [1, 2, 2, 3, 4, 5] (mantiene orden)
