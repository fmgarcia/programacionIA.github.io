    7. lgebra Lineal

    8. Broadcasting

    Broadcasting permite operar arrays de diferentes formas.

    # Escalar con array
arr = np.array([1, 2, 3])
print(arr * 2)  # [2 4 6]

# Array 1D con 2D
matriz = np.array([[1, 2, 3], [4, 5, 6]])
fila = np.array([10, 20, 30])

print(matriz + fila)
# [[11 22 33]
#  [14 25 36]]

# Columna con matriz
columna = np.array([[100], [200]])
print(matriz + columna)
# [[101 102 103]
#  [204 205 206]]

# Crear tabla de multiplicar con broadcasting
filas = np.arange(1, 11).reshape(10, 1)
cols = np.arange(1, 11)
tabla = filas * cols
print(tabla)

    Reglas de Broadcasting

    1. Si los arrays tienen diferente n煤mero de dimensiones, se a帽aden 1s a la izquierda.
    2. Arrays con tama帽o 1 en una dimensi贸n se estiran para coincidir.
    3. Si los tama帽os no coinciden y ninguno es 1, error.
    # Ejemplo: (3,) y (3, 3)

b = np.ones((3, 3))           # (3, 3)
# a se convierte en (1, 3) -> broadcasting a (3, 3)
