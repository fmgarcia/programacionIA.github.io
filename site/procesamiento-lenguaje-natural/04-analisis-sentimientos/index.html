    4.1. 쯈u칠 es el An치lisis de Sentimientos?

    Es el proceso computacional de identificar y categorizar opiniones expresadas en texto para determinar si la actitud del escritor hacia un tema es:

    • Positiva: "Me encanta este producto, es incre칤ble"
    • Negativa: "Terrible servicio, muy decepcionado"

    Niveles de An치lisis

    Nivel Descripci칩n Ejemplo
    Documento Sentimiento global del texto Rese침a completa de un producto
    Oraci칩n Sentimiento de cada oraci칩n Cada frase de una rese침a

    Entidad Sentimiento hacia entidades espec칤ficas Opiniones sobre diferentes marcas en un texto

    Tipos de Salida

    • Binaria: Positivo / Negativo
    • Ternaria: Positivo / Neutral / Negativo
    • Escala: Rating num칠rico (1-5 estrellas)
    • Emociones: Alegr칤a, Tristeza, Ira, Miedo, Sorpresa, Disgusto

    4.2. Enfoques para An치lisis de Sentimientos

    Enfoque Basado en L칠xico

    Utiliza diccionarios de palabras con polaridad predefinida.

    Ventajas:

    • No requiere datos de entrenamiento.
    • Interpretable y transparente.
    • Funciona bien en dominios generales.

    Desventajas:

    • No captura contexto ni negaciones bien.
    • Depende de la calidad del l칠xico.
    • Dificultad con sarcasmo e iron칤a.
    # Ejemplo simple de enfoque l칠xico

lexico_negativo = {'malo', 'terrible', 'horrible', 'peor', 'triste', 'odio', 'decepcionante'}

def analisis_lexico(texto):
    tokens = texto.lower().split()
    score = 0

    for token in tokens:
        if token in lexico_positivo:
            score += 1
        elif token in lexico_negativo:
            score -= 1

    if score > 0:
        return "Positivo"
    elif score < 0:
        return "Negativo"
    else:
        return "Neutral"

# Ejemplos
print(analisis_lexico("Este producto es excelente y genial"))  # Positivo
print(analisis_lexico("Terrible servicio, muy malo"))  # Negativo
print(analisis_lexico("El paquete lleg칩 ayer"))  # Neutral

    Enfoque Basado en Machine Learning

    Entrena clasificadores supervisados con datos etiquetados.

    Proceso t칤pico:

    1. Recolectar datos etiquetados (rese침as con ratings).
    2. Preprocesar texto (tokenizaci칩n, limpieza).
    3. Vectorizar (BoW, TF-IDF, embeddings).
    4. Entrenar clasificador (Naive Bayes, SVM, Logistic Regression).
    5. Evaluar y ajustar.

    Ventajas:

    • Captura patrones complejos.
    • Adaptable a dominios espec칤ficos.
    • Mejor rendimiento general.

    Desventajas:

    • Requiere datos etiquetados.
    • Puede no generalizar a otros dominios.

def analizar_sentimiento_textblob(texto):

# Ejemplos

    "Terrible experience, very disappointed",
    "The package arrived yesterday"
]

    print(f"   {resultado['sentimiento']} (pol: {resultado['polaridad']:.2f})")

    4.6. An치lisis de Sentimientos por Aspectos

    El Aspect-Based Sentiment Analysis (ABSA) identifica sentimientos hacia aspectos espec칤ficos de un producto o servicio.

    "La bater칤a del tel칠fono dura mucho, pero la c치mara es terrible"

Aspectos:

- c치mara  Negativo

    Implementaci칩n Simple

    'bater칤a': ['dura', 'excelente', 'incre칤ble', 'buena'],
    'c치mara': ['n칤tida', 'clara', 'buena', 'incre칤ble'],
    'pantalla': ['brillante', 'clara', 'grande', 'hermosa'],
    'dise침o': ['elegante', 'bonito', 'moderno', 'hermoso']
}

aspectos_negativos = {
    'bater칤a': ['corta', 'mala', 'terrible', 'poco'],
    'c치mara': ['borrosa', 'mala', 'terrible', 'p칠sima'],
    'pantalla': ['peque침a', 'oscura', 'mala'],
    'dise침o': ['feo', 'malo', 'anticuado']
}

def absa_simple(texto):
    """An치lisis de sentimiento por aspectos simple."""
    texto_lower = texto.lower()
    resultados = {}

    # Buscar aspectos y palabras de sentimiento cercanas
    for aspecto in aspectos_positivos.keys():
        if aspecto in texto_lower:
            sentimiento = "Neutral"

            # Buscar palabras positivas
            for palabra in aspectos_positivos[aspecto]:
                if palabra in texto_lower:
                    sentimiento = "Positivo"
                    break

            # Buscar palabras negativas
            for palabra in aspectos_negativos.get(aspecto, []):
                if palabra in texto_lower:
                    sentimiento = "Negativo"
                    break

            resultados[aspecto] = sentimiento

    return resultados

# Ejemplo

print(absa_simple(texto))
# {'bater칤a': 'Positivo', 'c치mara': 'Negativo'}

from collections import Counter

def monitorear_marca(menciones, classifier):
    """Analiza el sentimiento de menciones de una marca."""
    resultados = []

    for mencion in menciones:
        analisis = classifier(mencion)[0]
        resultados.append(analisis['label'])

        porcentaje = (count / total) * 100

    return resultados
