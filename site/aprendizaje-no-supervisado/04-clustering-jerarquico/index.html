    4.2. ExplicaciÃ³n MatemÃ¡tica

    Matriz de Distancias

    El clustering jerÃ¡rquico comienza calculando una matriz de distancias \(D\) de tamaÃ±o \(n \times n\):

    \[D_{ij} = d(x_i, x_j)\]

    Donde \(d\) es una funciÃ³n de distancia (tÃ­picamente Euclidiana).

    Criterios de Enlace (Linkage)

    La clave del clustering jerÃ¡rquico es cÃ³mo se calcula la distancia entre clusters. Existen varios criterios de enlace:

    1. Enlace Simple (Single Linkage) - "Vecino mÃ¡s cercano"

    Distancia entre los dos puntos mÃ¡s cercanos de cada cluster:

    \[d(C_i, C_j) = \min_{x \in C_i, y \in C_j} d(x, y)\]
    • âœ… Puede detectar clusters de formas arbitrarias
    • âŒ Sensible al "efecto cadena" (clusters elongados)

    2. Enlace Completo (Complete Linkage) - "Vecino mÃ¡s lejano"

    Distancia entre los dos puntos mÃ¡s lejanos de cada cluster:

    \[d(C_i, C_j) = \max_{x \in C_i, y \in C_j} d(x, y)\]
    • âœ… Produce clusters compactos de tamaÃ±o similar
    • âŒ Sensible a outliers

    3. Enlace Promedio (Average Linkage - UPGMA)

    Promedio de todas las distancias entre pares de puntos:

    \[d(C_i, C_j) = \frac{1}{|C_i| \cdot |C_j|} \sum_{x \in C_i} \sum_{y \in C_j} d(x, y)\]
    • âœ… Balance entre single y complete
    • âœ… Menos sensible a outliers

    4. Enlace de Ward (Ward's Method)

    Minimiza el incremento en la varianza total al fusionar clusters:

    \[d(C_i, C_j) = \sqrt{\frac{2|C_i||C_j|}{|C_i|+|C_j|}} ||\mu_i - \mu_j||\]

    Donde \(\mu_i\) y \(\mu_j\) son los centroides de los clusters.

    Equivalentemente, Ward minimiza:

    \[\Delta = \sum_{x \in C_i \cup C_j} ||x - \mu_{ij}||^2 - \sum_{x \in C_i} ||x - \mu_i||^2 - \sum_{x \in C_j} ||x - \mu_j||^2\]
    • âœ… Tiende a producir clusters esfÃ©ricos y de tamaÃ±o similar
    • âœ… Similar a K-Means pero jerÃ¡rquico
    • âŒ Asume clusters esfÃ©ricos

# EJEMPLO BÃSICO: Clustering JerÃ¡rquico con Dendrograma
# ============================================================

import matplotlib.pyplot as plt             # VisualizaciÃ³n

    linkage,        # Calcular el enlace jerÃ¡rquico
    dendrogram,     # Crear el dendrograma
    fcluster        # Obtener clusters del dendrograma
)

from sklearn.preprocessing import StandardScaler  # Escalado

# -------------------------------------------------------------

# -------------------------------------------------------------

    n_samples=50,       # 50 puntos (pequeÃ±o para visualizaciÃ³n clara)
    centers=4,          # 4 clusters
    cluster_std=0.60,   # DispersiÃ³n moderada
    random_state=42

# -------------------------------------------------------------

    metric='euclidean'  # MÃ©trica de distancia
)

print(f"[cluster_1, cluster_2, distancia, n_puntos]")

# 3. VISUALIZAR EL DENDROGRAMA
# -------------------------------------------------------------

    Z,
    truncate_mode='lastp',  # Mostrar los Ãºltimos p clusters fusionados
    p=20,                    # NÃºmero de clusters a mostrar
    leaf_rotation=90,        # Rotar etiquetas de hojas
    leaf_font_size=10,       # TamaÃ±o de fuente
    show_contracted=True     # Mostrar clusters contraÃ­dos
)

plt.ylabel('Distancia (Altura)', fontsize=12)

# Esta lÃ­nea indica dÃ³nde cortarÃ­amos para obtener cierto nÃºmero de clusters

            label=f'Corte en altura={corte}')

plt.tight_layout()
plt.show()

# -------------------------------------------------------------
# 4. OBTENER ASIGNACIONES DE CLUSTER
# -------------------------------------------------------------
# fcluster() "corta" el dendrograma para obtener clusters
# t: umbral de corte
# criterion: cÃ³mo interpretar t

axes[0].set_ylabel('Feature 2')
plt.colorbar(scatter1, ax=axes[0], label='Cluster')

# Clusters especificando k=4
scatter2 = axes[1].scatter(X[:, 0], X[:, 1], c=labels_k,
                           cmap='viridis', edgecolors='w', s=50)
axes[1].set_title('Clusters (k=4 especificado)')
axes[1].set_xlabel('Feature 1')
axes[1].set_ylabel('Feature 2')
plt.colorbar(scatter2, ax=axes[1], label='Cluster')

plt.tight_layout()
plt.show()

print("âœ… Cortando a diferentes alturas obtenemos diferentes nÃºmeros de clusters")

    4.7. Criterio de Enlace: Â¿CuÃ¡l Elegir?

    Criterio Forma de Clusters Sensibilidad a Outliers CuÃ¡ndo Usar
    Single Arbitraria, elongada Baja Detectar clusters de forma irregular
    Complete Compacta, esfÃ©rica Alta Clusters de tamaÃ±o similar
    Average Moderada Media Balance general
    Ward Compacta, esfÃ©rica Media Similar a K-Means, varianza mÃ­nima
